# Sim Companies (シムカンパニーズ) 日本語攻略データ・ガイド

## 概要 (About)
本リポジトリは、オンラインビジネスシミュレーションゲーム**「Sim Companies（シムカンパニーズ）」**の日本人プレイヤー向け攻略情報、および[公式日本語Wiki（@wiki）](https://w.atwiki.jp/simcompaniesjp/pages/1.html)のバックアップ・データ集です。
初心者向けの序盤の進め方から、会計・役員管理・各業界のサプライチェーンまで、日本語での詳細な解説をマークダウン形式でまとめています。

## コンテンツ一覧 (Contents)
AIアシスタントやユーザーが情報を検索しやすいよう、ジャンルごとにマークダウンで整理しています。

1.[🔰 初心者ガイド (Beginner's Guide)](jpwikibackups/beginnersguide_sc.md)
   - 初期構成（食品店＋畑）からの発展、実績ハントプレイのやり方
2.[📊 会計早見表 (Accounting Guide)](jpwikibackups/accountingguide_sc.md)
   - 損益計算書(P/L)の見方、投資回収期間(ROI/ROE)の計算、キャッシュフローの管理
3. [🏢 役員ガイド (Board Member Guide)](jpwikibackups/boardmemberguide_sc.md)
   - COO, CFO, CMO, CTOの雇うタイミング、経営負担率(管理費/AO)の計算式と対策
4. [🏭 業界ガイド (Industry Guide)](jpwikibackups/industryguide_sc.md)
   - 小売・生産・バランス・金融特化の構成解説、サプライチェーンの仕組み
5. [🤝 取引・市場ガイド (Market Guide)](jpwikibackups/marketguide_sc.md)
   - 小売価格・卸売価格・調達コストの考え方、契約（B2B）のコツ
6. [🏪 小売ガイド (Retail Guide)](jpwikibackups/retailguide_sc.md)

7.[🔬 リサーチ・研究ガイド (Research Guide)](jpwikibackups/researchguide_sc.md)

8.[🤖 ロボティクスガイド (Robotics/Specialization Guide)](jpwikibackups/specializationguide_sc.md)

9.[🎄 季節イベントガイド (Seasonal Events)](jpwikibackups/seasonalguide_sc.md)
   - ハロウィン、クリスマス（植林場システム）、夏・春のイベント解説
10.[💳 債券ガイド (Bonds Guide)](jpwikibackups/bondsguide_sc.md)
